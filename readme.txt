Acapela-VaaS service integration in your Wordpress website

== Description ==

This plugin allows you to install and use the Acapela-VaaS overlay on your Wordpress site.
The credentials you have to provide through the plugin settings will be used in an inserted script tag to connect to Acapela services, enabling text-to-speech features across your website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/acapela-vaas` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Acapela-VaaS screen to fill your credentials and configuration
